# My template using ChakraUi with NextJs.

###### A template for dashboard-type NextJs applications. Using a declarative interface library called ChakraUi.

**Install Project**

```
yarn
```

**Run Project Local**

```
yarn dev
```

**Run Project's Tests**

```
yarn test
```

**Build Project**

```
yarn build
```

**Run Project Production**

```
yarn start
```
